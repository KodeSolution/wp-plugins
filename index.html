Installation

The latest version of the each plugin can be downloaded from the gh-pages branch as a WordPress installable package. DO NOT click the Github "Download ZIP" button, this plugin is meant to be installed in the WordPress plugins directory and Github will not package the plugin correctly.

Installing plugins.

  1. Download the {each-plugin-name}.zip and unpack the archive.
  2. Upload {each-plugin-name} to the /wp-content/plugins/ directory.
  3. Activate the plugin through the 'Plugins' menu in WordPress.
